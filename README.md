# Wortspur

Ein deutsches Wortsuchspiel auf einem 4×4-Brett: Finde innerhalb von 3 Minuten
möglichst viele Wörter, indem du benachbarte Felder zu einem Pfad verbindest.

## Tech

Reines HTML, CSS und JavaScript &ndash; keine Frameworks, keine Abhängigkeiten,
kein Build-Schritt.

## Einschränkung

Es gibt **keine Wörterbuch-Validierung**. Geprüft wird nur, ob sich das Wort
auf dem Brett als gültiger Pfad bilden lässt &ndash; nicht, ob es ein echtes
deutsches Wort ist.
